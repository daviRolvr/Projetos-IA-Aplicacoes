# 🤖 Projetos de IA — Coleção de Aplicações com Inteligência Artificial

Bem-vindo! Este repositório reúne projetos desenvolvidos com foco em Inteligência Artificial, cobrindo desde agentes conversacionais até visão computacional e processamento de linguagem natural. Cada projeto é independente, organizado em sua própria pasta com documentação e código fonte.

Este repositório também faz parte do meu portfólio de projetos e estudos, criado para complementar meu aprendizado e apoiar minha entrada e evolução na carreira.

> Observação: novos projetos serão adicionados com o tempo. Esta página dá uma visão geral; para detalhes de cada projeto, acesse o README específico da pasta correspondente.

## 📦 O que você encontra aqui

- Documentação detalhada de cada solução

## 🧠 Projetos disponíveis

### 1) 🥦 IA Nutricionista — Agente Virtual de Nutrição

Breve descrição:
- Agente virtual avançado que une inteligência artificial com a ciência da nutrição para oferecer orientações alimentares personalizadas via Telegram.
- O "Nutrólogo Supremo" utiliza conhecimentos em bioquímica, metabolismo e diversas abordagens dietéticas, além de analisar imagens de refeições para estimar informações nutricionais.
- Registra peso e refeições, envia lembretes personalizados e gera relatórios semanais de progresso.

Tecnologias principais: Python, LangChain, OpenAI API, TinyDB, Pyrogram, Docker.
